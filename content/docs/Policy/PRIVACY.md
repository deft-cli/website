**Last Updated: June 2026**

This Privacy Policy explains how the Deft Project ("we", "us", or "our"), operated under the pseudonym **xntas**, handles information when you visit our website hosted on Cloudflare Pages and use the Deft CLI tool.

### 1. Information We Do Not Collect
Deft is a zero-dependency, local-first build tool. 
* **The Deft CLI Tool:** Does not collect, track, or transmit any personal data, source code, or telemetry to external servers. All compilation, caching, and dependency routing happen strictly on your local machine.
* **Personal Data:** We do not require registration, accounts, or any personal identifying information to use this website or the software.

### 2. Website Analytics and Infrastructure
Our website is built as a static site and hosted via Cloudflare. 
* We may use privacy-first website analytics (such as Cloudflare Web Analytics) that track non-identifying aggregate data (e.g., browser type, operating system, and page views) to improve performance. 
* This data is entirely anonymous and does not utilize tracking cookies or harvest personal IP addresses.

### 3. External Links and Source Code Repositories
Our website links to external source code repositories (e.g., GitHub). We are not responsible for the privacy practices or content of third-party platforms. When you checkout dependencies or sync indexes, you are subject to the privacy policies of those respective platforms.

### 4. Future Updates
As the Deft Project expands, we may introduce commercial services or paid enterprise features. If those changes require data collection, this Privacy Policy will be updated accordingly.
